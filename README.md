# Commands

## `chli gen`

command used to generate the desired framework files

```
USAGE
  $ chli gen

DESCRIPTION
  command used to generate the desired framework files

EXAMPLES
  $ chli gen
```

## `chli help [COMMAND]`

Display help for chli.

```
USAGE
  $ chli help [COMMAND] [-n]

ARGUMENTS
  COMMAND  Command to show help for.

FLAGS
  -n, --nested-commands  Include all nested commands in the output.

DESCRIPTION
  Display help for chli.
```
